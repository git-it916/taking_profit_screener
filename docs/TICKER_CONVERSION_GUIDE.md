# 티커 변환 가이드

한글 종목명 300개를 Bloomberg 티커로 한 번에 변환하는 방법입니다.

---

## 🎯 3가지 방법

### 방법 1: 복사 & 붙여넣기 (가장 간단!) ⭐
### 방법 2: 엑셀 파일 사용
### 방법 3: 텍스트 파일 사용

---

## ⭐ 방법 1: 복사 & 붙여넣기 (추천!)

### Step 1: 변환 프로그램 실행

```bash
python convert_tickers.py
```

또는 더블클릭:
```
convert_tickers.bat
```

### Step 2: 옵션 1 선택

```
변환 방법을 선택하세요:
  1: 직접 입력 (복사 & 붙여넣기)
  2: 파일에서 읽기 (.txt, .xlsx)
  3: 등록된 종목 목록 보기

선택 (1/2/3): 1
```

### Step 3: 종목명 붙여넣기

엑셀이나 다른 곳에서 종목명을 복사해서 붙여넣기:

```
삼성전자, SK하이닉스, LG에너지솔루션, 삼성바이오로직스, NAVER, 카카오
```

또는 줄바꿈으로:

```
삼성전자
SK하이닉스
LG에너지솔루션
삼성바이오로직스
NAVER
카카오
```

### Step 4: 입력 완료

- **Windows**: `Ctrl + Z` 누르고 엔터
- **Mac/Linux**: `Ctrl + D`

### Step 5: 결과 복사

```
================================================================================
변환된 Bloomberg 티커 (6개)
================================================================================

005930 KS, 000660 KS, 373220 KS, 207940 KS, 035420 KS
035720 KS

================================================================================

복사해서 start_bloomberg.py에 붙여넣기:
005930 KS, 000660 KS, 373220 KS, 207940 KS, 035420 KS, 035720 KS
```

➡️ 이걸 복사해서 `start_bloomberg.py`에 붙여넣기!

## 📊 방법 2: 엑셀 파일 사용

### Step 1: 엑셀 템플릿 열기

`ticker_mapping_template.xlsx` 파일 열기

### Step 2: 종목명과 티커 입력

| 종목명 | 티커 |
|--------|------|
| 삼성전자 | 005930 KS |
| SK하이닉스 | 000660 KS |
| LG에너지솔루션 | 373220 KS |
| ... | ... |
| (300개까지) | (300개까지) |

### Step 3: 저장

`my_stocks.xlsx`로 저장

### Step 4: 변환 실행

```bash
python convert_tickers.py
```

옵션 2 선택:
```
선택 (1/2/3): 2

입력 파일 경로: my_stocks.xlsx
```

### Step 5: 결과 확인

자동으로 Bloomberg 티커로 변환됨!

## 🔧 추가 종목 등록하기

기본 제공되지 않는 종목은 직접 추가할 수 있습니다.

### 옵션 A: 매핑 파일 사용 (권장)

1. **`ticker_mapping_template.xlsx` 복사**
2. **종목명과 티커 추가**:

| 종목명 | 티커 |
|--------|------|
| 내가 원하는 종목 | 123456 KS |
| 또 다른 종목 | 789012 KS |

3. **`my_mapping.xlsx`로 저장**

4. **변환 시 매핑 파일 지정**:
```
사용자 정의 매핑 파일이 있습니까?
매핑 파일 경로: my_mapping.xlsx
```

### 옵션 B: 코드에 직접 추가

`src/ticker_converter.py` 파일 수정:

```python
KOREAN_STOCK_MAP = {
    # 기존 종목들...
    '삼성전자': '005930 KS',

    # 여기에 추가
    '내가 원하는 종목': '123456 KS',
    '또 다른 종목': '789012 KS',
}
```

## ⚠️ 문제 해결

### 문제 1: "변환 실패 종목"이 나옴

```
⚠️  변환 실패 종목 (3개):
  - 알 수 없는 종목1
  - 알 수 없는 종목2
  - 알 수 없는 종목3
```

**해결**:
1. Bloomberg에서 정확한 티커 확인
2. `ticker_mapping_template.xlsx`에 추가
3. 다시 변환

### 문제 2: 종목명 철자가 다름

예: "삼성전자" vs "삼성전자주식회사"

**해결**:
매핑 파일에 둘 다 추가:
```
삼성전자, 005930 KS
삼성전자주식회사, 005930 KS
```

### 문제 3: 티커 형식이 잘못됨

올바른 형식:
- ✅ `005930 KS` (공백 포함)
- ❌ `005930KS` (공백 없음)
- ❌ `005930` (시장 코드 없음)
